?Cross-Device Sync Setup Guide

1
Create a free Supabase account
Go to supabase.com and click Start your project. Sign up with GitHub or email. It's completely free for personal use.
2
Create a new project
Click New project, choose a name (e.g. "bookmarks"), set a database password, and select the region closest to you. Wait ~2 minutes for setup to complete.
3
Create the sync table
In your project, go to SQL Editor and run this SQL:
create table bookmark_sync ( user_id uuid primary key references auth.users(id), bookmarks text, folders text, updated_at timestamptz default now() ); alter table bookmark_sync enable row level security; create policy "own data" on bookmark_sync using (auth.uid() = user_id) with check (auth.uid() = user_id);
4
Disable email confirmation (recommended)
Go to Authentication → Settings and turn off Enable email confirmations. This allows you to sign in immediately after creating an account without verifying your email.
5
Sign in on this device
On PC: click the account button (circle icon, top right). On mobile: open Prefs → Account & Sync. Enter your email and password, then click Sign in. First time? Use Create account to register. Once signed in, your session is saved automatically.
6
Sync across devices
After signing in, use Push to cloud to upload your bookmarks. On your other devices, open this HTML file in a browser, sign in with the same email and password — your session will be remembered automatically. Use Pull from cloud to download your bookmarks.
7
Mobile (iPhone / Android)
Save this HTML file to your phone (via iCloud, Google Drive, email, etc.) and open it in Safari or Chrome. Sign in via Prefs → Account & Sync. On iOS, you can add it to your Home Screen via Safari → Share → Add to Home Screen.
